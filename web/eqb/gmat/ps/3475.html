The function f is defined for each positive three-digit integer n by
A.8
B.9
C.18
D.20
E.80

题干给出了一个f(n)的定义，同理可知，f(m) = (2^x)(3^y)(5^z)。

因为f(m) = 9f(v)，即f(m) = 3^2 f(v)。转换一下可见：f(m) / f(v) = 3^2.

也就是说f(m)和f(v)的差别就在于3的幂了（因为其他都没约掉了）。

又f(m)的3的幂是y,且f(m)和f(v)相差的幂为2，所以f(v)的3的幂就是y-2（因为y-(y-2)=2）。

又y代表的是一个三位数的十位，十位要乘以10，所以m与v之间相差的数值就是2*10=20.
