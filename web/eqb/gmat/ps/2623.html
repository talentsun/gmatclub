If x and y are positive integers and 1 + x + y + xy = 15, what is the value of x + y?
A.3
B.5
C.6
D.8
E.9

(x+y)+xy=15-1

(x+y)+xy=14

14= 1+13 = 2+12 = 3+11 = 4+10 = 5+5 = 6+8 = 7+7

因為x和y都是整數 而且x*y不會是質數 因此只剩 4+10 和 6+8 兩組

4 = 1+3 = 2+2    而 1*3或者2*2都不會等於10

10 = 1+9 = 2+8 = 3+7 = 4+6 = 5+5  而1*9或2*8或3*7或4*6或5*5都不會等於4

6 = 1+5 = 2+4 =3+3   而2*4=8    所以x+y=6而x*y=8    (x+y)+xy=6+8=14  正解

8= 1+7 = 2+6 = 3+5 = 4+4  而1*7或2*6或3+5或4*4都不會等於6
