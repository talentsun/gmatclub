If –4 ＜ x ＜ 7 and –6 ＜ y ＜ 3, which of the following specifies all the possible values of xy?
A.–42 ＜ xy ＜ 21
B.–42 ＜ xy ＜ 24
C.–28 ＜ xy ＜ 18
D.–24 ＜ xy ＜ 21
E.–24 ＜ xy ＜ 24

If –4 ＜ x ＜ 7 and–6 ＜ y ＜ 3,

  1. 从x>-4和y>-6来看，xy<24；从x<7,y<3来看，xy<21;
  2. 因为要找出xy 的所有可能的结果，因此xy的最小值应大于-6*7=-42, 最大值应小于24.所以选择B
